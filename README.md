# Command Notes

This repository includes a comprehensive set of commands (~90%) for a smooth workflow in:

- **Git** – version control and GitHub workflow
- **Docker** – container management and Docker Compose
- **Linux** – basic and essential terminal commands

---

## Repository Contents

- `github-workflow.md` – Git commands and workflow notes
- `docker-workflow.md` – Docker and Docker Compose commands
- `linux-commands.md` – Basic Linux commands for developers

---

## Purpose

The goal of this repository is to provide a **quick reference** for daily development tasks, streamline workflows, and help avoid common mistakes when using Git, Docker, and Linux.
